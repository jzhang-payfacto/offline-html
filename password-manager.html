Operation and privacy

This page uses JavaScript running entirely within your web browser to encrypt and decrypt your vault client-side. It makes no network connections, so your data and master password never leave the browser. This can be verified by reviewing the page source or monitoring your browser's network activity; a Content-Security-Policy embedded in the page additionally instructs the browser to block all network requests. Passwords copied to the clipboard are automatically overwritten after 30 seconds (best effort — this page must still be open).

Running this page offline

This page is self-contained — all HTML, CSS and JavaScript are in this single file and it uses no external libraries. Save it as an .html file and open it in your browser to run it offline.
